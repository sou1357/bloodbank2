# Assets Directory

This directory should contain the following image assets for the Expo app:

## Required Assets

1. **icon.png** (1024x1024 px)
   - App icon displayed on the home screen
   - Should feature the LifeFlow logo (blood drop)
   - Background: Red (#DC2626)

2. **splash.png** (1242x2436 px)
   - Splash screen shown when app launches
   - Should have LifeFlow branding
   - Background: Red (#DC2626)

3. **adaptive-icon.png** (1024x1024 px)
   - Android adaptive icon
   - Foreground layer with logo
   - Background: Red (#DC2626)

4. **favicon.png** (48x48 px)
   - Web favicon (if running on web)

## Creating Assets

You can create these assets using:
- Design tools: Figma, Adobe Illustrator, Canva
- Online generators: makeappicon.com, appicon.co
- Or use placeholder images for development

## Temporary Solution

For development/testing, you can use Expo's default assets by removing the asset references from `app.json`, or create simple placeholder images with any image editor.

## Colors

- Primary Red: #DC2626
- White: #FFFFFF
- Background: #F9FAFB
